# RFM Customer Segmentation Analysis

## Project Overview
Analyzed 4000+ customers from UK-based e-commerce store
using RFM (Recency, Frequency, Monetary) methodology.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI (Interactive Dashboard)
- Dataset: Kaggle - Online Retail UCI

## Key Findings
- 6 customer segments identified
- Champion customers (25%) generate 40%+ of total revenue
- At-Risk customers need re-engagement campaign
- Potential Revenue Recovery: Rs.5.2 Lakhs

## Project Steps
1. Data Cleaning & EDA
2. RFM Calculation
3. Customer Segmentation
4. SQL Analysis
5. Power BI Dashboard
